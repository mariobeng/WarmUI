# WarmUI 适老化助手 — 浏览器插件

## 概述

**WarmUI 适老化助手** 是一款面向老年用户本人的 Chrome 浏览器插件（Manifest V3），属于 WarmUI 适老化设计体系的**终端用户工具层**。它通过实时增强网页的视觉呈现、交互行为和认知可理解性，让老年用户能够更轻松、更自信地使用任意 Web 产品。

它与 **WarmUI 老龄化模拟器**（面向开发者的工具）形成完整闭环：

| 插件         | 目标用户             | 核心目的             | 作用方向                       |
| ------------ | -------------------- | -------------------- | ------------------------------ |
| 老龄化模拟器 | 开发者、设计师       | 模拟障碍，建立同理心 | 让页面**变差**（模拟衰退）     |
| 适老化助手   | 老年用户本人、照护者 | 消除障碍，改善体验   | 让页面**变好**（增强可访问性） |

## 设计哲学

### 三层能力模型

继承 WarmUI 核心设计理念，适老化助手从三个层次增强网页体验：

#### 🎯 第一层：感官补偿（视觉增强）

解决老年人最常见的视力衰退问题——老花、对比度敏感度下降、色觉辨别力减弱。

- **智能字体放大**：语义级缩放，区分标题/正文/按钮/标注，而非简单全局 zoom
- **高对比度配色**：注入医学验证的高对比度配色方案，支持多套色盲主题
- **阅读模式**：简化页面布局，隐藏干扰元素，优化行间距和字间距
- **语音朗读**：选中文本后 TTS 朗读，支持语速和音调调节

#### 🖐️ 第二层：操作辅助（触觉增强）

解决老年人手抖、精细动作控制下降导致的误触问题。

- **点击热区扩大**：自动扩大可点击元素的交互区域至 44px 以上
- **防误触保护**：对确认/删除等关键操作增加二次确认
- **滚动优化**：增大滚动条宽度，平滑滚动体验
- **焦点增强**：高亮当前焦点元素，优化 Tab 导航顺序

#### 🧠 第三层：认知辅助

解决老年人面对复杂信息时的认知负担——理解困难、注意力分散、迷失方向。

- **大白话翻译**：检测长难句，提供简化版本悬浮气泡
- **页面引导**：自动识别核心操作区，呼吸动画高亮提示
- **简化导航**：生成页面内容大纲和快速跳转目录
- **专注模式**：自动隐藏广告、弹窗、自动播放视频等干扰元素
- **迷路救援**：浮动"救生圈"按钮，一键回到首页或上一步

### 核心设计原则

1. **隐私优先**：所有处理在本地完成，调用云端时明确告知且不存储
2. **渐进增强**：默认不改变任何页面，用户主动开启后才生效
3. **场景预设**：提供"大字号"、"简洁"、"语音优先"等一键场景模式
4. **配置持久化**：设置通过 `chrome.storage.sync` 跨设备同步
5. **尊重原站**：非侵入式注入，关闭插件后页面完全恢复原状

## 技术架构

### 目录结构

```
packages/elder-plugin/
├── manifest.json                   # Manifest V3
├── package.json                    # 包配置
├── vite.config.js                  # Vite 构建配置
├── popup/                          # 弹出控制面板
│   ├── popup.html                  # 适老化UI
│   ├── popup.css                   # 大字号高对比度样式
│   └── popup.js                    # 面板交互逻辑
├── content/                        # 内容脚本（核心引擎）
│   ├── content.js                  # 主脚本：协调所有增强模块
│   ├── enhancers/                   # 增强模块
│   │   ├── index.js                # 模块注册导出
│   │   ├── font-enhancer.js        # 字体放大
│   │   ├── contrast-enhancer.js    # 对比度/配色
│   │   ├── layout-enhancer.js      # 阅读模式
│   │   ├── click-enhancer.js       # 点击热区/防误触
│   │   ├── speech-enhancer.js      # 语音朗读
│   │   └── guide-enhancer.js       # 页面引导/认知辅助
│   └── presets.js                  # 场景预设配置
├── background/
│   └── service-worker.js           # Service Worker
└── icons/                          # 插件图标
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

### 技术选型

| 层面       | 选择                          | 理由                           |
| ---------- | ----------------------------- | ------------------------------ |
| Manifest   | V3                            | Chrome 最新标准，更安全更高效  |
| 构建工具   | Vite                          | 与 WarmUI 现有工具链一致       |
| CSS 注入   | 动态 `<style>` + `!important` | 覆盖任意网站原有样式，非侵入式 |
| 配置持久化 | `chrome.storage.sync`         | 跨设备同步用户偏好             |
| TTS        | Web Speech API                | 浏览器原生支持，零额外依赖     |
| AI 翻译    | 可选的云端 API（按需开启）    | 隐私优先，本地兜底             |
| popup UI   | 纯 HTML/CSS，超大字体         | 面板本身就要适老化             |

### 增强模块架构

每个增强器（Enhancer）是独立模块，遵循统一接口：

```javascript
// 增强器标准接口
class BaseEnhancer {
  // 唯一标识
  static id = "enhancer-name";
  // 显示名称
  static label = "增强器名称";
  // 启用增强
  enable(config) {}
  // 禁用增强
  disable() {}
}
```

主脚本 `content.js` 通过消息机制接收 popup 的配置指令，动态调用各个增强器的 `enable()` 方法，并将配置持久化到 `chrome.storage`。页面刷新后自动恢复已有配置。

### 与模拟器插件的对比

| 维度     | 老龄化模拟器 (plugin/)                         | 适老化助手 (elder-plugin/)                     |
| -------- | ---------------------------------------------- | ---------------------------------------------- |
| 注入方向 | filter 模糊、clipPath 管状视野、animation 手抖 | fontSize 放大、contrast 增强、padding 扩大热区 |
| 认知模拟 | 隐藏图标、替换术语、减慢速度                   | 简化文案、引导提示、专注模式                   |
| 交互模式 | 勾选后被动感受                                 | 根据需求主动调节                               |
| 配置存储 | `chrome.storage.local`                         | `chrome.storage.sync`（跨设备同步）            |
| popup UI | 标准字号，开发者工具风格                       | 大字号高对比度，适老化设计                     |

## 功能详情

### 场景预设

插件内置多个一键场景模式，每个模式是多个增强功能的组合：

| 预设名称        | 包含增强                              | 适用场景                     |
| --------------- | ------------------------------------- | ---------------------------- |
| 📖 大字号模式   | 字体放大 150% + 高对比度 + 行间距加大 | 视力正常衰退的老花用户       |
| 🧘 简洁模式     | 阅读模式 + 隐藏广告 + 专注模式        | 容易被干扰、感到混乱的用户   |
| 🗣️ 语音优先模式 | 自动朗读 + 语音导航 + 大字            | 视力严重下降或偏好听觉的用户 |
| 🌙 夜间模式     | 低蓝光配色 + 柔和对比度 + 降低亮度    | 夜间浏览或光敏感用户         |

### 视觉增强 (Phase 2)

#### 智能字体放大

- 语义级缩放：heading/body/button/caption 按不同比例放大
- 三级强度：125% / 150% / 200%
- 伴随调整：行高 1.6→1.8→2.0，字间距 0→0.5px→1px

#### 高对比度配色

- 复用 `warm-theme.js` 的 5 套配色主题
- 自动检测页面背景色，智能选择最适配的对比方案
- 支持滤镜级增强：`contrast(1.2) brightness(1.1)`

#### 阅读模式

- 隐藏广告、侧边栏、浮层等干扰元素
- 优化正文区域：最大宽度、居中显示、统一字体
- 为图片添加边框和放大功能

### 操作增强 (Phase 3)

#### 点击热区扩大

- 自动检测 `<a>`、`<button>`、`<input>` 等可交互元素
- 通过伪元素 `::after` 扩大点击区域（不影响布局）
- 最小保证 44×44px 的触控目标（WCAG 2.1 标准）

#### 防误触保护

- 检测"删除"、"确认"、"提交"等高风险操作
- 注入自定义确认弹窗
- 可配置两次确认或长按确认

### 认知辅助 (Phase 4)

#### 页面引导

- 自动识别页面中主要操作按钮
- 呼吸动画光晕高亮
- 语音提示"请点击这里"

#### 大白话翻译

- 检测长句（超过 30 字）和复杂词汇
- 长按/悬停显示简化版解释气泡
- 可接入大模型 API 或本地规则引擎

## 构建与安装

### 前置条件

| 条件          | 说明            |
| ------------- | --------------- |
| Node.js       | >= 18           |
| npm           | 随 Node.js 自带 |
| Chrome 浏览器 | 建议最新稳定版  |

### 构建插件

```bash
# 在项目根目录安装依赖
cd e:\my\warm-ui
npm install

# 构建适老化助手插件
cd packages/elder-plugin
npm run build
```

构建产物位于 `packages/elder-plugin/dist/`：

```
dist/
├── content.js            # 内容脚本
├── enhancers.js          # 增强模块
├── presets.js            # 预设配置
├── service-worker.js     # Service Worker
└── popup/
    └── popup.html        # 弹出面板
```

### 安装到 Chrome

1. 打开 `chrome://extensions/`
2. 开启右上角"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `packages/elder-plugin` 目录

### 使用方法

1. 安装后，Chrome 右上角出现 🧓 图标
2. 点击图标打开控制面板
3. 选择预设场景或单独调节各项增强
4. 设置自动保存，刷新页面后自动恢复
