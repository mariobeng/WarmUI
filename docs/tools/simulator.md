# 老龄化模拟器浏览器插件

让开发者一键体验老年人的视、触、知世界。

## 安装

1. 打开 Chrome 扩展管理页面 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `packages/plugin` 目录

## 模拟模式

| 类别 | 功能 | 说明 |
|------|------|------|
| 👓 视觉 | 模糊视觉 | 模拟老花眼 |
| 👓 视觉 | 黄化视觉 | 晶状体变黄 |
| 👓 视觉 | 管状视野 | 视野收窄 |
| 👓 视觉 | 对比度下降 | 色彩分辨困难 |
| 🖐️ 触觉 | 手抖（页面） | 页面整体抖动 |
| 🖐️ 触觉 | 手抖（光标） | 光标不规则偏移 |
| 🖐️ 触觉 | 精细动作减弱 | 按钮点击困难 |
| 🧠 认知 | 信息处理变慢 | 响应延迟 |
| 🧠 认知 | 注意力易分散 | 视觉干扰 |
| 🧠 认知 | 认知降级 | 图标隐藏、专业术语替换 |
