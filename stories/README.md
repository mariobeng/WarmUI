# Stories — 组件设计故事与使用场景

每个 WarmUI 组件背后都有一段故事。本目录记录每个组件为何而设计、在什么场景下使用、以及如何真正帮助到老年用户。

## 目录结构

```
stories/
├── button.md        # 按钮 — 从"怕点错"到"放心按"
├── text.md          # 文字 — 从"看不清"到"读得懂"
├── dialog.md        # 对话框 — 从"不知所措"到"温和引导"
├── guide.md         # 引导 — 从"迷路了"到"下一步"
├── note.md          # 便签 — 从"记不住"到"家人的提醒"
├── rescue.md        # 救援 — 从"慌了"到"有我在"
├── speech.md        # 语音 — 从"看不见"到"听得见"
├── translator.md    # 翻译 — 从"看不懂"到"说人话"
├── praise.md        # 夸赞 — 从"不确定"到"我真棒"
└── buddy.md         # 协助 — 从"帮帮我"到"一起搞定"
```
